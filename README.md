# Google-Play-Store-EDA-Analysis
Exploratory Data Analysis (EDA) of the Google Play Store dataset using Python, Pandas, Matplotlib, and Seaborn. This project analyzes app categories, ratings, reviews, installs, content ratings, and trends through statistical analysis and visualizations to uncover meaningful business insights.


Proejct Overview:- 
This project performs Exploratory Data Analysis (EDA) on the Google Play Store dataset to understand app trends, user engagement, ratings, installs, and category-wise performance.


Technologies Used :- 
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook


Key Insights :- 
- Games are among the most installed app categories.
- Most applications are free.
- Ratings are concentrated between 4.0 and 4.5.
- Certain categories generate significantly higher installs.
- User reviews positively correlate with app popularity.
